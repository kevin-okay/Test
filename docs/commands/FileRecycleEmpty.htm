FileRecycleEmpty

清空回收站.

FileRecycleEmpty [, DriveLetter]

参数

DriveLetter

如果省略, 则清空所有驱动器的回收站. 否则, 请指定驱动器字母, 例如 C:\

ErrorLevel

[v1.1.04+] 此命令失败时会抛出异常. 想了解更多信息, 请参阅 运行时错误.

如果遇到问题则 ErrorLevel 被置为 1, 否则为 0.

备注

此命令要求安装 MS Internet Explorer 4 或更高版本.

相关

FileRecycle, FileDelete, FileCopy, FileMove

示例

FileRecycleEmpty, C:\
