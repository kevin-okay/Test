EnvMult

设置变量的值为它自身乘以给定的值。同义于：Var *= Value。

EnvMult, Var, Value

参数

Var

要运算的 变量 名.

值

任何整数, 浮点数或 表达式.

备注

此命令等同于这种简写形式：Var *= Value。

如果 Var 或 Value 其中一个为空或不以数字开头，那么在计算中它会被视为 0（不过用在表达式内部时例外，例如 Var := X *= Y）。

如果 Var 或 Value 都为小数, 那么最后的结果将为浮点数, 而格式由 SetFormat 决定.

相关

EnvAdd, EnvSub, EnvDiv, SetFormat, 表达式, If var is [not] type, SetEnv, 按位运算 (Transform)

示例

EnvMult, MyCount, 2 
MyCount *= 2  ; 等同于上面的语句
