FileGetTime

获取文件或文件夹的时间戳。

FileGetTime, OutputVar [, Filename, WhichTime]

参数

OutputVar

用来保存获取的 YYYYMMDDHH24MISS 格式的日期时间的变量名. 这个时间是您本地时间, 不是 UTC/GMT.

Filename

目标文件或文件夹的名称, 如果未指定绝对路径则假定在 %A_WorkingDir% 中. 如果省略, 则使用最内层 文件循环 的当前文件.

WhichTime
要获取的时间戳类别:
M = 修改时间 (此参数省略时这是默认的)
C = 创建时间
A = 上次访问时间

ErrorLevel

[v1.1.04+] 此命令失败时会抛出异常. 想了解更多信息, 请参阅 运行时错误.

如果遇到问题则 ErrorLevel 被置为 1, 否则为 0.

A_LastError 被设置为操作系统 GetLastError() 函数返回的结果.

备注

请参阅 YYYYMMDDHH24MISS 了解日期和时间的说明.

相关

FileSetTime, FormatTime, If var is [not] type, FileGetAttrib, FileSetAttrib, FileGetSize, FileGetVersion, 文件循环, EnvAdd, EnvSub

示例

FileGetTime, OutputVar, C:\My Documents\test.doc  ; 默认获取修改时间.
FileGetTime, OutputVar, C:\My Documents\test.doc, C  ; 获取创建时间.
